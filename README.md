## 🌐 Sobre a 44CODE

A 44CODE entrega tecnologia sob medida — do design da arquitetura ao deploy seguro em produção. Atuamos em:

- Engenharia de Software (Web / SaaS multi-tenant)
- DevOps & DevSecOps (segurança integrada ao ciclo de desenvolvimento)
- Cibersegurança (formação contínua via Programa Hackers do Bem / RNP)
- Automação inteligente com integração a modelos de IA

## 🚀 Tecnologias utilizadas

| Camada | Tecnologia |
| --- | --- |
| Framework | TanStack Start v1 (React 19 + SSR) |
| Roteamento | TanStack Router (file-based) |
| Build tool | Vite 7 |
| Linguagem | TypeScript (strict) |
| Estilo | Tailwind CSS v4 + Design tokens (oklch) |
| UI | shadcn/ui + Radix UI + lucide-react |
| Validação | Zod |
| Notificações | Sonner |
| Deploy | Cloudflare Workers (Wrangler) |

## 📂 Estrutura do projeto

\`\`\`
src/
├── assets/            # Imagens e logos
├── components/        # Componentes reutilizáveis (Header, Footer, WhatsAppButton, ui/)
├── hooks/             # Hooks customizados
├── lib/               # Utilitários (cn, helpers)
├── routes/            # Páginas (file-based routing)
│   ├── __root.tsx     # Layout global
│   ├── index.tsx      # Home
│   ├── empresa.tsx    # Sobre a empresa
│   ├── servicos.tsx   # Serviços oferecidos
│   ├── portfolio.tsx  # Portfólio
│   ├── sobre.tsx      # Perfil profissional
│   └── contato.tsx    # Formulário de contato
├── router.tsx         # Configuração do router
└── styles.css         # Design system (tokens, gradientes, sombras)
\`\`\`

## 🛠️ Rodando localmente

Pré-requisitos: **Node.js 20+** e **bun** (ou npm/pnpm).

\`\`\`bash
bun install        # Instalar dependências
bun run dev        # Servidor de desenvolvimento (http://localhost:5173)
bun run build      # Build de produção
\`\`\`
